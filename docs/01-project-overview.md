# プロジェクト概要

## 目的

単人プレイの中で、Tank が敵の強攻撃を受け止め、支援キャラクター Hikari の負荷を管理しながら反撃機会を作る 3D アクション RPG プロトタイプです。

## 現在の主なループ

1. 敵をターゲットする。
2. 基礎攻撃または範囲基礎攻撃でダメージを与える。
3. 敵の詠唱攻撃や AoE を見て、被ダメージ軽減スキルを使う。
4. 正しく強攻撃を受けると Guard Resonance が発生し、Hikari の光負荷が下がる。
5. 条件を満たすと Radiant Riposte による反撃機会を得る。
6. 戦闘後にドロップを拾得し、装備で攻撃力や最大 HP を強化する。
