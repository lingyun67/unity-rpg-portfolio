# AI Assisted Development Workflow

## 使用方針

開発中は AI を、実装方針の整理、コードリーディング、バグ原因の切り分け、コメントやドキュメントの整形に利用しています。

## 自分で判断する部分

- Unity の Inspector 参照、Prefab / Scene の実際の状態
- 実行時ログと Play Mode での再現確認
- AI が前提にしたクラスやフィールドが実在するかどうか
- 既存システムの責務分担を壊していないか

## このリポジトリでの整理方針

この公開用リポジトリでは、コードの処理部分は変更せず、コメントの読みやすさと説明資料の整理のみを行っています。Inspector 表示文字列や Debug.Log 文字列は C# のコード内容に含まれるため変更していません。
