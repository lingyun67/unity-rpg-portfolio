# Unity Action RPG Portfolio

このリポジトリは、個人で開発している Unity アクション RPG プロトタイプから、公開可能なコードと技術資料のみを抜粋したポートフォリオです。

元の Unity プロジェクトには第三者アセット、Scene / Prefab、未公開のゲーム内容が含まれるため非公開にしています。このリポジトリには、公開可能なコードサンプルと説明資料のみを掲載しています。

## プロジェクト概要

本プロジェクトは、単人プレイで MMORPG の Tank 役割に近い責任感を体験することを目指した 3D アクション RPG プロトタイプです。プレイヤーは敵の詠唱攻撃や範囲攻撃を見極め、被ダメージ軽減スキル、反撃、装備構築、Hikari の光負荷管理を使って戦闘を進めます。

## 収録内容

- `code-samples/skills/`：プレイヤースキル、実行時状態、基礎攻撃、反撃、スキルバー UI
- `code-samples/hikari/`：Hikari の自動回復、光負荷、守護共鳴、溢光反震
- `code-samples/enemy/`：敵 AI、敵スキル、詠唱バー、ドロップ処理、敵全体制御
- `code-samples/inventory-equipment/`：アイテムデータ、インベントリ、装備、戦闘ステータス
- `docs/`：各システムと各ファイルの説明

## コメント整理について

公開用として読みやすくするため、元ファイル内の詳細コメントは削除し、各ファイル冒頭に日本語の要約コメントを追加しています。C# の処理、フィールド名、メソッド名、文字列リテラル、Inspector の Header / Tooltip、Debug.Log の内容は変更していません。
